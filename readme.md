## About The Project

![Screen Shot](https://s8.uupload.ir/files/screenshot_from_2024-03-05_11-57-47_m0yj.png)

üõ° Nola is a script that will back up MongoDB databases that run by docker.
It has 2 methods:
* üñê Manual back up: this method will back up database and upload to a host by FTP account.
* üî® Automatic back up: in this method, you can set interval hours to get back up and upload to host by FTP account.

**Note that you need an FTP account to upload back up files.**
